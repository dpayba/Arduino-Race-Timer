# arduino-race-timer
Drag race timer designed for a 2 car race system.

Written in C++

Functions:
- Tracks reaction speed of both lanes
- Detects any false starts 
- Logs time of 1st and 2nd thirds
- Logs final time
- System at end turns on LED of winner

Parts Used:
- Arduino
- Photoresistors
- Male to male wires
- Male to female wires
- Buzzer
- LCD Display
